  • Google翻译是如何把深度学习“塞进”手机的

    为了达到翻译的即时性，我们也做了大量的优化工作，而且优化和反复调试的工作量很大。此外，为了改善计算性能，在某些数学运算上，也需要手工重度调优。我们充分利用SIMD（Single instruction, multiple data，单指令流多数据流），这是一种数据并行的模式，来挖掘智能多核手机的并行潜能。同时，我们还优化矩阵乘法，提高矩阵运算的局部性，使之在各级缓存（Cache）中，提高Cache的命中率。

Image repositories

这里提供了一些国内的镜像，以解决部分国外镜像较慢的问题。后续将添加自己的镜像。

  • NPM镜像

    淘宝NPM镜像：https://registry.npm.taobao.org

    使用方法：

    $ npm install something --registry=https://registry.npm.taobao.org

    或者在配置文件里修改默认镜像

  • pip镜像

    豆瓣：http://pypi.douban.com/simple

    V2EX：http://pypi.v2ex.com/simple

    使用方法（以豆瓣镜像为例）：

    $ pip install something -i http://pypi.douban.com/simple

    或者在配置文件里修改默认镜像 （linux的文件在~/.pip/pip.conf，windows在%HOMEPATH%\pip\pip.ini）

    修改或添加以下语句：

    [global]
    index-url = http://pypi.douban.com/simple

  • Maven/Gradle镜像

    开源中国Maven仓库：http://maven.oschina.net/content/groups/public/

    使用方法：

    修改Maven安装目录下conf文件夹的 settings.xml 文件

    <mirrors>
    <!-- mirror | Specifies a repository mirror site to use instead of a given 
    repository. The repository that | this mirror serves has an ID that matches 
    the mirrorOf element of this mirror. IDs are used | for inheritance and direct 
    lookup purposes, and must be unique across the set of mirrors. | -->
    	<mirror>
    		<id>nexus-osc</id>
    		<mirrorOf>*</mirrorOf>
    		<name>Nexus osc</name>
    		<url>http://maven.oschina.net/content/groups/public/</url>
    	</mirror>
    </mirrors>

  • 其它国内镜像

    清华大学 TUNA 镜像源：http://mirrors.tuna.tsinghua.edu.cn/
